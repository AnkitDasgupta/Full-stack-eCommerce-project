A production-ready multi-vendor eCommerce web application designed to support scalable online marketplaces, enabling multiple vendors to manage their products while providing customers with a seamless shopping experience.

The platform is built using modern, industry-standard technologies including Spring Boot (backend), React with TypeScript (frontend), and JWT-based authentication to ensure secure and efficient communication between client and server.

Implemented a robust role-based access control system supporting Admin, Vendor, and Customer roles, ensuring secure authorization and controlled resource access. Designed and developed RESTful APIs to enable smooth frontend-backend integration.

Key features include:

🛍️ Product Management System for vendors to add, update, and manage inventory
🛒 Shopping Cart & Order Processing for seamless purchasing flow
💳 Payment Integration to handle secure transactions
🔐 Authentication & Authorization using JWT for enhanced security
📦 Order Tracking System for real-time status updates
🤖 Chatbot Integration to improve user engagement and support
