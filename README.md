Developed a production-ready multi-vendor eCommerce
web application that allows multiple sellers to manage
products while customers can browse, purchase, and track
orders. The platform was built using industry-standard
technologies including Spring Boot, React, TypeScript,
& JWT.
Designed and implemented secure authentication and
authorization, role-based access for admins, vendors, and
customers, and RESTful APIs for seamless frontend
backend communication. Developed core features such as
product management, order processing, cart system,
payment flow, and chatbot integration.
